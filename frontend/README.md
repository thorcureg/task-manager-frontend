# Task Manager Frontend

This is the frontend repository for a task management application developed as a solo project using React.

The project is inspired by modern project management platforms such as ClickUp and focuses on providing a clean and responsive user experience for managing tasks. Users can create, edit, organize, and track tasks through an interactive interface that communicates with a backend REST API.

The goal of this project is to demonstrate practical frontend development skills, including component-based architecture, state management, API integration, and user interface design.

## Tech Stack

* React
* JavaScript
* REST API
* CSS / Tailwind CSS
